# VideoEditor

Video Editor. License: GPL v2. 

Available in English and Spanish

Convert, divide, join, crop, rotate videos. 

Extract music 

Put 'Watermark'. 

Change the volume (up/down or mute), etc. 

Simple video editor with the power of FFMPEG
